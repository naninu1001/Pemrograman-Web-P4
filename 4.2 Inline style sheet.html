Belajar pemograman web CSS

CSS adalah aturan untuk mengendalikan beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. CSS bukan merupakan bahasa pemrograman, melainkan sebuah bahasa styling atau bahasa markup yang digunakan bersama dengan nHTML untuk mengatur tampilan visual dokumen web.

CSS adalah singkatan dari Cascending Style Sheets.
